# ⚔️ SolRage

**SolRage** is a decentralized warrior economy built on Solana.  
Collect NFT cards, battle in a fast-paced arena, and earn $SOLRAGE by staking and playing.

## 🧱 Features

- 🔗 Solana + Phantom wallet integration
